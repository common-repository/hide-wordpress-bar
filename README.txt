Plugin to hide and display the admin bar in an easy way

== Description ==

With it you can manage the time when you would like to display or display the WordPress admin bar
If you have any comments or questions, please use the support forum:
https://wordpress.org/support/plugin/auto-hide-admin-bar

= Feedback =
If you want, you can drop me a line @[odigoribeiro](http://twitter.com/odigoribeiro) on Twitter.

== Installation ==

1. Upload `hide-wordpress-bar` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Ready. Now show an icon as a button where you go can show or hide the administration bar of WordPress

For a manual installation via FTP:

1. Upload the `ab-backend-search` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' screen in your WordPress admin area


To upload the plugin through WordPress, instead of FTP:

1. Upload the downloaded zip file on the 'Add New' plugins screen (see the 'Upload' tab) in your WordPress admin area and activate.

== Frequently Asked Questions ==
